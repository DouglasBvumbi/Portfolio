## Image Optimization Tips

1. **Compress images** before uploading to reduce file sizes
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Aim for file sizes under 300KB

2. **Use appropriate formats**:
   - JPG for photos and complex images
   - PNG for images with transparency or text
   - WebP for modern browsers (optional)

3. **Responsive images**:
   - The website will automatically scale images
   - Provide high-quality originals (2x size) for retina displays

4. **Alt text**: Images already have descriptive alt text in HTML
